# IT 设备维护指南

## 一、笔记本电脑常见问题与解决方案

### 1.1 开机无反应
**可能原因**：电池耗尽、电源适配器故障、主板故障  
**处理步骤**：
1. 检查电源适配器是否正常连接，指示灯是否亮起
2. 尝试拔掉电源，长按电源键 15 秒后重新接入电源开机
3. 如仍无法开机，在系统中提交报修工单

### 1.2 运行缓慢
**可能原因**：内存不足、磁盘空间不足、恶意软件  
**处理步骤**：
1. 关闭不必要的后台程序
2. 清理临时文件和回收站
3. 运行杀毒软件进行全盘扫描
4. 如问题持续，联系 IT 部门评估是否需要硬件升级

### 1.3 屏幕显示异常
**可能原因**：显卡驱动问题、屏幕物理损坏、线缆接触不良  
**处理步骤**：
1. 重启电脑并更新显卡驱动
2. 外接显示器测试是否为屏幕硬件问题
3. 如确认屏幕损坏，提交报修工单

## 二、打印机常见问题

### 2.1 无法打印
**处理步骤**：
1. 检查打印机是否开机且在线
2. 确认打印队列中没有卡住的任务
3. 重新安装打印机驱动
4. 检查网络连接（网络打印机）

### 2.2 打印质量差
**处理步骤**：
1. 检查墨盒/硒鼓剩余量
2. 运行打印机自带的喷头清洗程序
3. 检查纸张是否受潮
4. 如需更换耗材，通过系统申请领用

## 三、网络设备维护

### 3.1 网络不通
**处理步骤**：
1. 检查网线是否连接牢固
2. 查看交换机端口指示灯状态
3. 尝试重启交换机（需在非工作时间操作）
4. 联系网络管理员检查 VLAN 和 ACL 配置

### 3.2 WiFi 信号弱
**处理步骤**：
1. 确认是否在 AP 覆盖范围内
2. 检查是否有信号干扰源
3. 联系网络管理员调整 AP 位置或增加 AP

## 四、服务器维护规范

### 4.1 日常检查项
- 每日检查 CPU、内存、磁盘使用率
- 每周检查系统日志和安全日志
- 每月进行一次完整的数据备份验证
- 每季度进行一次硬件巡检（电源、风扇、硬盘指示灯）

### 4.2 紧急处理
- 服务器宕机：立即通知 IT 经理和相关业务方
- 磁盘告警：优先进行数据备份，然后排查故障
- 安全入侵：立即断网隔离，通知安全团队

## 五、报修流程

### 5.1 提交报修
1. 登录 NexAsset 系统
2. 找到对应资产
3. 点击"报修"按钮
4. 描述故障现象

### 5.2 维修处理
1. IT 部门接到工单后，1 个工作日内响应
2. 评估维修方案和预计时间
3. 维修完成后在系统中点击"完成维修"
4. 设备回到"在库"状态，可重新分配

### 5.3 维修记录
所有维修记录都会保存在系统审计日志中，包括：
- 报修时间和维修完成时间
- 故障描述和维修方案
- 维修成本（如有）
